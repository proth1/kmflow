KMFlow Agent has been installed successfully.

The KMFlow Task Mining Agent is now installed on your Mac and will start automatically each time you log in.

Next steps

  1. Complete the onboarding wizard
    Open KMFlow Agent from your Applications folder. The onboarding wizard will guide you through connecting to your engagement backend and confirming your capture preferences.
  2. Grant Accessibility permission
    The agent requires Accessibility access to observe UI activity. When prompted, click Open System Settings and enable KMFlow Agent under Privacy & Security → Accessibility.
    Note: If your IT administrator has pre-approved access via MDM, this step may already be complete.
  3. Verify the agent is running
    Look for the KMFlow menu bar icon to confirm the agent is active and connected to your engagement.

Need help?

  • Refer to the KMFlow Agent user guide at docs.kmflow.ai
  • Contact your engagement lead or IT administrator
  • Email support: support@kmflow.ai

To uninstall the agent, run the uninstall script included in the KMFlow documentation package, or follow the instructions at docs.kmflow.ai/uninstall.
