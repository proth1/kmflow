Welcome to KMFlow Task Mining Agent

This installer will set up the KMFlow Task Mining Agent on your Mac. The agent observes UI activity to automatically capture process intelligence for your KMFlow engagement.

What will be installed

  • KMFlow Agent.app — installed to /Applications
  • LaunchAgent — a background service entry installed to ~/Library/LaunchAgents/com.kmflow.agent.plist so the agent starts automatically at login
  • Application Support folder — local evidence buffer at ~/Library/Application Support/KMFlowAgent/

Before you continue

  • macOS 13.0 (Ventura) or later is required.
  • Approximately 500 MB of free disk space is required.
  • After installation you will be prompted to grant Accessibility permission in System Settings. The agent cannot observe UI activity without this permission.
  • Your IT administrator may have pre-approved permissions via MDM, in which case no manual steps are needed.

To remove the agent at any time, run the uninstall script provided in the KMFlow documentation, or contact your IT administrator.
